    📝 Resumo do Projeto: Sistema de Cadastro Escolar
    Este projeto é uma aplicação de console desenvolvida em C# (.NET) que demonstra os 
    fundamentos da Programação Orientada a Objetos (POO) aplicada a um cenário de gestão acadêmica.
    
    🏗️ Estrutura e Funcionalidades
    Encapsulamento e Classe de Domínio (Aluno.cs): Criação de uma classe Aluno 
    com propriedades automáticas para armazenar: Nome, Idade, Sala, Professor 
    e Matéria.
    
    Uso de um Construtor para garantir que o objeto seja instanciado com todos os 
    dados obrigatórios.
    
    Método ExibirDados() para centralizar a lógica de formatação e saída de 
    informações.
    
    Lógica Principal (Program.cs):
    Coleções Dinâmicas: Utilização de uma List<Aluno> para armazenar múltiplos 
    cadastros em memória durante a execução.
    
    Estrutura de Repetição: Implementação de um laço while que permite ao usuário 
    cadastrar diversos alunos de forma contínua.
    
    Entrada e Saída (I/O): Captura de dados via teclado (Console.ReadLine) e 
    tratamento básico de tipos (conversão de string para int).
    
    Iteração de Dados: Uso do laço foreach para percorrer a lista e exibir o relatório 
    final de todos os objetos instanciados.
    
    🚀 Tecnologias Utilizadas
    Linguagem C#
    .NET SDK
    VS Code como ambiente de desenvolvimento (IDE)
